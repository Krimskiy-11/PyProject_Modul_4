# Проект E-commerce.

## Описание:

E-commerce — электронная торговля, или электронная коммерция.

## Цель проекта:

На данном этапе работы мы не будем реализовывать систему платежей, 
однако подготовим всё для того, чтобы у нас появилось ядро для интернет-магазина.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Krimskiy-11/PyProject_Modul_4.git
```

## Тестирование:

1. Тестирование всех функций в пакете tests
```
pytest tests/
```
2. Тестирование функций по отдельности:
```
pytest tests/test_category.py
pytest tests/test_product.py
pytest tests/test_iterator.py
```
3. Проверка покрытия тестами:
```
pytest --cov
```
